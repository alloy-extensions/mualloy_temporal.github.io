MuAlloy Temporal

MuAlloy Temporal is a mutation testing tool for the Alloy modeling language that generates mutants, generates a mutant-killing test suite and performs mutation testing.

Source Code Open source code base for the latest release. Includes the Alloy4Fun temporal model benchmark.
